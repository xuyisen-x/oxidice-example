# How to use it?

You can visit (https://xuyisen-x.github.io/oxidice-example/)[https://xuyisen-x.github.io/oxidice-example/] to see a live demo of this project.

If you want to run this project locally, please follow the steps below:

## 1. Update Submodules

```bash
git submodule update --init --recursive
```

## 2. Install Dependencies

```bash
npm install
```

## 3. Run the Development Server

```bash
npm run dev
```

## 4. Build for Production

```bash
npm run build
```

then you can find the built files in the `dist` directory.
